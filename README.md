# CharityHelp

**CharityHelp** is a desktop-based application that helps manage multiple charities within a single platform. It offers a streamlined solution for handling charity databases and employee records efficiently.

## 🛠 Tech Stack

- **Frontend**: Python Tkinter (GUI)
- **Backend**: MySQL

## 🚀 Features

- Admin and Employee Login System
- Charity Database Management
- Employee Management System

## 🧩 Installation

> ⚠️ This project runs locally. Ensure you have Python and MySQL installed on your machine.

1. **Clone the repository**  
   ```bash
   git clone https://github.com/adityaacharya7/CharityHelp.git
   cd CharityHelp
````

2. **Set up the MySQL database**

   * Import the provided `.sql` file (if available) into your MySQL server.
   * Update the database credentials in the Python files if necessary.

3. **Run the application**

   ```bash
   python main.py
   ```

## 📘 How to Use

Simply run the Python file (`main.py` or the entry-point file). Use the login interface to sign in as Admin or Employee and start managing charities and users.
